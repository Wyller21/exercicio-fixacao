Dentro da lista abaixo estarão listadas tags de Html e suas principais funções.

1 - <a> :
É usado para indicar um link para outra página.

2 - <body> :
É uma tag que indica onde vai estar o corpo do documento html incluindo todos os seus dados.

3 - <br> :
Usado para realizar uma quebra de linha.

4 - <div> :
Cria uma seção dentro do código html, usado para criar seguimentos e sepações.

5 - <footer> :
Cria a seção de rodapé onde são adicionados dados de contato e informações finais da página

6 - <h1>-<h6> :
É usado para definir a importância dos títulos dentro do corpo html indo de h1 até h6, sendo o h1 o de maior importância.

7 - <head> :
Tag onde é armazenado os metadados, configurações da página html.

8 - <img> :
Tag usada para adicionar uma imagem ao arquivo html, sendo a imagem provinda de um espaço local ou link externo.

9 - <ul> :
Esta tag cria uma lista em formato não ordenado (para indicar os dados dentro da lista usar a tag <li>).

10 - <p> :
Esta tag cria um parágrafo onde normalmente pe adicionado texto mas pode ser usado para links e fotos também.
